# مسار أساسيات TypeScript

الشيفرة المصدرية الخاصة تطوير تطبيق تعقب النفقات في مسار أساسيات TypeScript ضمن دورة "تطوير التطبيقات باستخدام JavaScript" المقدمة من أكاديمية حسوب
---

## 🚀 خطوات تشغيل التطبيق

1. تثبيت الحزم اللازمة
   npm install

2. إعداد متغيرات البيئة
   - قم بإنشاء ملف .env في المجلد الرئيسي
   - أضف المتغيرات المطلوبة (مفاتيح قاعدة البيانات، JWT، إلخ)

3. تشغيل التطبيق
   npm run dev
   
   سيعمل التطبيق على http://localhost:3000

---

## 📱 عن التطبيق

تتبع النفقات هو تطبيق ويب حديث ومتكامل يساعد المستخدمين على إدارة شؤونهم المالية الشخصية من خلال واجهة سهلة الاستخدام ومتعددة اللغات.

---

## ✨ المميزات الرئيسية

### 💰 إدارة مالية متكاملة
- ✅ تتبع النفقات والإيرادات اليومية
- ✅ تصنيف المعاملات المالية لتنظيم أفضل
- ✅ عرض أنماط الإنفاق والسجل المالي

### 📊 تصور البيانات والرسوم البيانية
- 📈 رسوم بيانية تفاعلية توضح توزيع النفقات
- 📉 رؤى مرئية لعادات الإنفاق حسب الفئة
- 📊 تحليل الإنفاق الشهري والسنوي

### 🔐 أمان وتسجيل الدخول
- 🔑 نظام تسجيل وتسجيل دخول للمستخدمين
- 🔒 تشفير كلمات المرور لحماية البيانات
- 👤 تتبع نفقات مخصص لكل مستخدم

### 🌐 دعم متعدد اللغات
- 🇸🇦 العربية
- 🇬🇧 الإنجليزية
- 🇫🇷 الفرنسية
- 🔄 دعم الكتابة من اليمين لليسار للغة العربية
- 🔀 تبديل سلس بين اللغات أثناء الاستخدام

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| Next.js 13 | إطار العمل الرئيسي لتطوير الواجهات |
| MongoDB | قاعدة البيانات لتخزين المعلومات |
| TypeScript | لكتابة كود آمن ومنظم |
| Chart.js | إنشاء الرسوم البيانية التفاعلية |
| JWT | المصادقة وحماية الجلسات |
| next-intl | دعم التعدد اللغوي |
